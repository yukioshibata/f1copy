- f1copy.exe ver 0.9 概要

* 基本機能
  - 起動時 以下を画面中央に表示、2秒後に消去する ※表示はセンタリング

  ```画面表示
  =================================================
  F1COPY Ver X
  =================================================
  F1 -> Ctrl+C
  F2 -> Ctrl+V
  CapsLock -> Ctrl
  ScrollLock -> CapsLock
  ```

  - 終了時 タイトルとバージョンを表示して、2秒後に消去する。※表示はセンタリング

  ```画面表示
  =================================================
              F1COPY Ver X
  =================================================
  ```

  - F1 Key が押されたとき、Ctrl+C を押したときと同じ動作をする
  - F2 Key が押されたとき、Ctrl+V を押したときと同じ動作をする
  - CapsLock キーを Ctrl Key として動作する
  - CapsLock + F1 は 元々の F1 Key として動作する(F2も同様)
  - ScrollLock キーが押されたときは、CapsLock Key の動作をする

* 実行プログラム f1copy.exe
  - 通常起動、常駐しているときは、タスクトレイにアイコンを表示する。アイコンをダブルクリックすると、プログラムを終了する。
  - -install PC起動時に自動起動するようタスクスケジューラに登録して起動する。管理者権限で実行されていない場合は、管理者権限で再実行する。インストール後、プログラムを常駐開始する。
  - -uninstall タスクスケジューラの登録を削除して、プログラムの常駐を終了する

* プログラム
  - C++で作成する
  - Windows APIを使用する
  - Windows11 64bitで動作する
  - VisualStudio2022Community で作成する
  - f1copy.ico をアイコンとして利用する

* 開発時の注意
  - 自動でコマンド（run_command）を実行する際には、先頭に chcp 65001 を実行して、文字コードをUTF-8にする
  - PowerShell用のエンコード指定を挟んで実行する
  - splash window は少しおしゃれにしてください
